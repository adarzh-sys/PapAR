
Unity AR Starter Project for iPhone SE 2020

Features:
- AR Foundation setup for iOS (ARKit)
- Plane detection
- Tap to place a cube
- Cube turns red on "simulated sensor alert"

To Run:
1. Open this project in Unity 6.1 (or 2022.3 LTS)
2. Go to File > Build Settings > Switch to iOS
3. Player Settings > XR Plugin Management > iOS > Enable ARKit
4. Build and open the Xcode project
5. Set your Apple Developer team
6. Run on your connected iPhone SE 2020

Let me know if you want to link this to live sensor data or add CSV reading logic.
